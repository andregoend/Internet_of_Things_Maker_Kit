<img src="/images/12_Web Client.PNG" height="400">

### Modul-modul yang dibutuhkan pada proyek ini:
* ESP8266 IoT V2.0 (1).

Apabila tidak terdapat kesalahan dan koneksi sukes, maka akan ditampilkan ke software Terminal.
